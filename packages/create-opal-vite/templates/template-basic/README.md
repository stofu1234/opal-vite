# Opal + Vite

This project was scaffolded with `create-opal-vite`.

## Getting Started

```bash
# Install dependencies
npm install
bundle install

# Start dev server
npm run dev
```

## Project Structure

```
├── app/
│   ├── javascript/
│   │   └── application.js   # JavaScript entry point
│   ├── opal/
│   │   └── application.rb    # Opal entry point
│   └── styles.css            # Application styles
├── index.html                # Main HTML
├── vite.config.ts            # Vite configuration
├── package.json
└── Gemfile
```
