Opal is a Ruby to JavaScript source-to-source compiler. It comes with an implementation of the Ruby corelib and stdlib, allowing you to write frontend code in Ruby.
Stimulus is a modest JavaScript framework for the HTML you already have. It connects JavaScript objects to HTML elements using data attributes, providing structure without complexity.
Combining Stimulus with Opal lets you write clean, expressive Ruby code for your frontend controllers while leveraging Stimulus's HTML-first approach. Vite provides fast HMR during development.

Overview

This example demonstrates how to build reusable UI components using Stimulus controllers written in Ruby via Opal. Each component follows Stimulus conventions: targets, values, actions, and CSS classes.

Features

  • Write Stimulus controllers in Ruby
  • Full access to Stimulus Values, Targets, and Classes APIs
  • Action parameters for flexible event handling
  • StimulusHelpers for DOM manipulation without raw JS
  • Hot Module Replacement via Vite

Usage

Each controller inherits from StimulusController and includes StimulusHelpers. Define targets, values, and classes as class-level attributes, then implement action methods that Stimulus will call from your HTML.
